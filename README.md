# Tiger365 ID - Online Cricket Betting ID Website

A complete Next.js + Tailwind CSS website for Tiger365 ID, an online cricket betting ID provider.

## Tech Stack
- **Next.js 14** with App Router
- **Tailwind CSS** for styling
- **TypeScript**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Sections
1. **Hero** - Big headline with mobile app mockup
2. **Sports Showcase** - Cricket, Football, Betting, Interface cards
3. **Our Services** - 6 service cards grid
4. **About** - Platform description with banner
5. **Team** - Support team + PLAY MORE section
6. **Cricket Features** - 3 feature blocks
7. **Service Cards** - 24/7 Support, Payments, Bonus
8. **Payment & Support** - Two-column layout
9. **FAQ** - Accordion Q&A
10. **Footer** - Links + WhatsApp float button
